# 加点要素
- スザンヌ、バニー、ティーポット、マルチメッシュの描画
- 原神公式が配布しているナヒーダというキャラのモデルの描画
- 複数モデルの描画(上記のモデル群の描画)
- 音声の再生
- コントローラー操作
- テキスト(フォント)の描画

# 操作
- ImGui上にて操作可能
- オブジェクト群というタブに音声やオブジェクトをいじるための部分があります
 - 「音声」というツリーにて音声の再生ができます
 - 「オブジェクト」というツリーにて、コンボ部分でオブジェクトを選択すると下のDragFloatなどの部分で値がいじれます
 - 「テキスト」というツリーにて、テキストの入力ができ、そのテキストが画面上のフォントに反映されます
- コントローラー操作は、ImGui上の「コントローラー入力」というタブで入力の値が見れるようになっています
